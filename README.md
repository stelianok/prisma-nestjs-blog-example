

## Description

This is a simple blog created with NestJS and Prisma 

## Tech Stack

- Typescript
- Docker
- NestJS
- Prisma
- PostgreSQL

---

## REST API Documentation

you can access the documentation in http://localhost:3000/api

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
